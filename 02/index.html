Розмітка заголовків

Заголовки впорядковують текст формуючи його візуальну структуру. При використанні заголовків необхідно враховувати їх ієрархію. В заголовки не можна вкладати інші елементи.
<h1> - заголовок найвищого рівня, на сторінці рекомендовано використовувати не більше одного разу.
<h2>, <h3>, <h4> <h5>, <h6> - заголовки нижчих рівнів розташовані за спаданням їх важливості

Синтаксис

<h1> Заголовок 1-го </h1> 
<h2> Заголовок 2-го</h2>
<h3> Заголовок 3-го</h3> 
<h4> Заголовок 4-го</h4> 
<h5> Заголовок 5-го</h5> 
<h6> Заголовок 6-го</h6>

Результат

Заголовок 1-го

Заголовок 2-го

Заголовок 3-го

Заголовок 4-го

Заголовок 5-го
Заголовок 6-го

Теги для форматування тексту

<strong> - вказує на важлівість фрагменту тексту. Зазвичай виділяє текст напівжирним

Синтаксис

Результат

Тег визначає важливість визначеного

<em> - вказує на логічне посилення фрагменту тексту, уточнює зміст виразу. Зазвичай виділяє тукст курсивом.

Синтаксис

Результат

HTML - головний інструмент веб-розробника

<small> - зменшує розмір шрифту на одиницю порівняно із звичайним текстом

Синтаксис

HTML - <small>зменшує </small> розмір шрифту

Результат

HTML - зменшує розмір шрифту

Деякі символи зарезервовано в HTML. Якщо ви використовуєте в тексті знаки менше ( <) або більше (>), браузер може змішати їх з тегами. Символьні сутності використовуються для відображення зарезервованих символів у HTML.

  • &nbsp; ( ) - нерозривний пробіл:
  • &copy; (©)- знак copyright
  • &trade; (™) - знак торгової марки
  • &lt; (<) - знак "меньше"
  • &gt; (>) - знак "більше"

Списки

Тег <li> визначає елемент списку. Тег <li> використовується всередині впорядкованих списків (<ol>), неупорядкованих списків (<ul>).
Тег <ul> визначає невпорядкований (маркований) список.

Синтаксис

HTML - <ul>
<li>Кофе</li>
</li>Чай</li>
<li>Молоко</li>
</ul>

Результат

HTML -

  • Кофе
  • Чай
  • Молоко

Тег <ol> визначає впорядкований список. Упорядкований список може бути цифровим або алфавітним.

Синтаксис

HTML - <ol>
<li>Кофе</li>
</li>Чай</li>
<li>Молоко</li>
</ol>

Результат

HTML -

  1. Кофе
  2. Чай
  3. Молоко

Зображення

Тег <img> використовується для вбудовування зображення на сторінку HTML. Зображення технічно не вставляються на веб-сторінку; зображення посилаються на веб-сторінки. Тег створює простір для утримання зображення, на яке посилається.
Тег має два обов'язкових атрибута:

  • src -Вказує шлях до зображення
  • alt - Вказує альтернативний текст для зображення, якщо зображення з якихось причин не може бути відображенo

Синтаксис

HTML - <img src="butterfly.png" alt="butterfly">

Результат

HTML -
