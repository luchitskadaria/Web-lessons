Основні властивості CSS для роботи з текстом

Властивості font-size

Властивості font-size змінює розмір шрифта.
Розмір шрифту задається за допомогою значення та одиниць виміру (em, px, pt, %)

Синтаксис

CSS

   p {
    font-size:20px;

    }

HTML

<p> Hello world!</p>

Результат:

Hello world

Властивість font-weight

Властивість font-weight встановлює насиченість шрифту
Може приймати наступні значення:
  • normal - нормальна насиченість, те саме, що і 400
  • bold - напівжирний текст, те саме, що і 700
  • lighter - змінює насиченість відносно бітьківського стилю в меншу сторону
  • bolder - змінює насиченість відносно бітьківського стилю в більшу сторону
  • 100, 200, 300, 400, 500, 600, 700, 800 900 - цифрові значення насиченості, які відрізняються від стандартних

Синтаксис

CSS

    bold{
        font-weight: bold;
    }
    .thin{
        font-weight: 100;
    }
    .max{
        font-weight: 900;
    }

HTML

    <p>
        <span class = 'bold'> Напівжирний текст</span>,
        <span class = 'thin'> найтонший текст</span>,
        <span class = 'max'> найтовщий текст</span>.
    </p>

Результат:

Напівжирний текст, найтонший текст, найтовщий текст.

Властивості font-style

Властивості font-style визначає стиль шрифту для тексту.
Може приймати наступні значення:
  • normal - відображає звичайний стиль шрифту. Це за замовчуванням
  • italic - відображає стиль шрифту курсивом
  • oblique - відображає похилий стиль шрифту

Синтаксис

CSS

    .normal {
        font-style: normal;
    }
    
    .italic {
        font-style: italic;
    }
    
    .oblique {
        font-style: oblique;
    }

HTML

    <p>
        <span class="normal">Це абзац, нормальний. </span>
        <span class="italic">Це абзац, курсив. </span>
        <span class="oblique">Це абзац, косий. </span>
    <p>

Результат:

Це абзац, нормальний.
Це абзац, курсив.
Це абзац, косий.

Властивість font-family

Властивість font-family визначає шрифт для елемента.

Синтаксис

CSS

    .font-family-Times {
        font-family: "Times New Roman", Times, serif;
    }
        
    .font-family-Arial {
        font-family: Arial, Helvetica, sans-serif;
    }

HTML

    <p class="font-family-Times">Це абзац, показаний шрифтом Times New Roman.</p>

    <p class="font-family-Arial">Це абзац, показаний шрифтом Arial.</p>

Результат:

Це абзац, показаний шрифтом Times New Roman.
Це абзац, показаний шрифтом Arial.

Властивості color

Властивість color звизначає колір тексту.

Синтаксис

CSS

    .red {
        color: LightGreen ;
    }

    .green {
        color: #CD5C5C;
    }

    .blue {
        color: rgb(30, 144, 255);;
    }

HTML

<pre>
    <span class="green"> Це абзац зеленого кольору </span> 
    <span class="red">Це абзац зеленого кольору </span>
    <span class="blue">Це абзац синього кольору </span>
</pre>

Результат:

Це абзац зеленого кольору
Це абзац зеленого кольору
Це абзац синього кольору

Властивості text-transform

Властивість text-transform контролює використання великих літер у тексті.
  • none - Без використання великих літер. Текст відображається таким, яким він є. Це за замовчуванням
  • capitalize - Перетворює перший символ кожного слова у верхній регістр
  • uppercase - Перетворює всі символи у верхній регістр
  • lowercase - Перетворює всі символи в нижній регістр

Синтаксис

CSS

    .upper {
        text-transform: uppercase;
        }
        
    .lower {
        text-transform: lowercase;
        }
        
    .capit {
        text-transform: capitalize;
        }

HTML


    <pre>
    <span class="green"> Це абзац зеленого кольору </span> 
    <span class="red">Це абзац зеленого кольору </span>
    <span class="blue">Це абзац синього кольору </span>
    </pre>

Результат:


text-transform: uppercase;
text-transform: lowercase;
text-transform: capitalize;

Властивість text-decoration

Властивість text-decoration визначає прикрасу, додану до тексту, і є скороченою властивістю для:
  • text-decoration-line(обов'язково)
  • text-decoration-color
  • text-decoration-style

Синтаксис

CSS

            
    .overline {
        text-decoration: overline;
    }
    
    .line-through {
        text-decoration: line-through;
    }
    
    .underline {
        text-decoration: underline;
    }
    
    .underline-overline {
        text-decoration: underline overline;
    }

HTML


    <pre>
    <span class="green"> Це абзац зеленого кольору </span> 
    <span class="red">Це абзац зеленого кольору </span>
    <span class="blue">Це абзац синього кольору </span>
    </pre>

Результат:


        text-decoration: overline
        text-decoration: line-through;
        text-decoration: underline
        text-decoration: underline overline
